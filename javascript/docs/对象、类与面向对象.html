# 构造函数

构造函数与普通函数唯一的区别就是调用方式不同，除此之外，构造函数也是函数。

任何函数只要使用new操作符调用就是构造函数。

# 构造函数的问题

其定义的方法会在每个实例上都创建一遍。会带来不同的作用域链和标识符解析，但创建新Function实例的机制是一样的。

都是做一样的事，没必要定义两个不同的Function实例，况且this对象可以吧函数与对象的绑定推迟到运行时。
